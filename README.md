# Introduction

Project made to learn Astro with Tailwind CSS.


## 🏗️ How to get started?

```
git clone 

cd astro-portfolio

```

## 📂 Project Structure


## 🧞 Commands

All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm run dev`             | Starts local dev server at `localhost:4321`      |
